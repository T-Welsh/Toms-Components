## Data
Takes in data in the following format...

```

$card_grid_data = (object)[
    'id' => "unique-id",
    'heading' => "HTML block containing section heading/blurb",
    'cards' => [An array of card objects]
]

```
- id --unique id for targenting specific instance of component
- heading --Sets the content for the section heading 
- cards --Array of html blocks that will display in the carousel
