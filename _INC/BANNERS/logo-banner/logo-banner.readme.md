# Logo Banner v1.0.0

## Overview
Adds a banner that display a simgle logo with a link. Intended to highlight membership of organisations such as checkatrade
- z-index group: 

## Data
Takes in data in the following format...

```

$logo_banner_data = (object)[
    'id' => 'banner-2',
    'logo_url' => "../_LIBS/images/logoipsum-215.svg",
    'logo_alt' => "image alt text",
    'background_color' => 'transparent',
    'button_url' => "#"
];

```
- id --unique id for targenting specific instance of component
- logo_url --Sets the logo image
- logo_alt --sets the alt text of the logo
- background_color -- set the banner color, will default to --clr-primary if not passed to component
- button_url --Linked address to go to on button click
